Blacklist means everything is disabled for the user who is blacklisted.

Blacklisted servers can't have the bot. If a bot joins a server which is blacklisted, it sends blacklist-server message and leaves the server.
If bot was already in a server before it was blacklisted, then it sends blacklist message in a channel and leaves the server.

Blacklist command (alias: bl)

if no argument then wrong usage message
add <user id or user mention or username>

list (no further arguments, if provided then ignore)
enable (no further arguments, if provided then ignore)
disable (no further arguments, if provided then ignore)

Arguments in blacklist-server command:

add <server id>

list (no further arguments, if provided then ignore)
enable (no further arguments, if provided then ignore)
disable (no further arguments, if provided then ignore)

blacklist remove - removes a server from blacklist list
blacklist list - lists the blacklist list

blacklist-server remove - removes a server from blacklist-server list
blacklist-server list - lists the blacklist-server list
blacklist-server enable - globally enables the blacklist-server
blacklist-server disable - globally disables the blacklist-server
